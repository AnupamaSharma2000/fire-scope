🔥 About FireScope

FireScope is a real-time global wildfire intelligence dashboard powered by live NASA FIRMS satellite fire detections. It visualizes fire activity on an interactive 3D globe and 2D map, computes fire danger indices, and surfaces high-priority alerts.

Data Sources

  • NASA FIRMS — Fire Information for Resource Management System. MODIS and VIIRS satellite fire detections updated every 3–12 hours. firms.modaps.eosdis.nasa.gov
  • Open-Meteo — Free open-source weather API for current conditions. open-meteo.com
  • Natural Earth — Country boundary data for geocoding. naturalearthdata.com

Key Metrics

  • FRP (Fire Radiative Power) — Measured in Megawatts, indicates the radiant heat output of a fire.
  • Brightness — Brightness temperature in Kelvin detected by the satellite sensor.
  • Confidence — Algorithm confidence: Low / Nominal / High.
  • Daynight — Whether the detection was during the day (D) or night (N) overpass.

Technology Stack

  • Globe.gl — 3D WebGL globe rendering
  • Leaflet.js — 2D interactive map
  • Chart.js + D3.js — Analytics visualizations
  • NASA FIRMS API — Live satellite fire data
  • Open-Meteo API — Real-time weather
